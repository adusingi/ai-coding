The program arc

From training
to a working product.

Phase 1 — Training
Weeks 1–8 · Remote · English

Intensive AI-Assisted Engineering

The full SDLC with AI: Prototype → Design → Build → Deploy → Support. Using Claude Code on a real project from day one. You will not finish this phase without something that runs.

Gate — Selection
End of week 8

A Real User, a Real Problem

Submit a field visit report. Show one real conversation with a future user. Ideas that solve real problems — especially for rural, Kinyarwanda-speaking users — move forward. This is not a quota. It is a filter for evidence.

Phase 2 — Product
3 months · $20/month Claude Pro

Build Beyond MVP

3 months of Claude Pro to build a working product with real users. Products enter the Mobayilo portfolio, promoted to investors, NGOs, and government programs. Graduates become mentors for the next cohort.
